# Banking System Project

This is a C project based on a basic banking system.  
It has options to create an account, deposit money, withdraw money and show the balance.

## Features
- Create Account  
- Deposit Amount  
- Withdraw Amount  
- Check Balance  

## How to Run
First compile the code using gcc: gcc -std=c11 -Wall -Wextra -Wpedantic -Iinclude src/main.c src/bank.c -o main
Then run the executable: ./main
## Folder Structure
- src → contains all .c files  
- include → contains header file  
- docs → for documentation  
- assets → other files  
- sample_input.txt → example input
